This package supports displaying transcriptions of observations, including any annotations that have been created on top of the transcriptions. The main elements of functionality in the package are:
  • Display transcriptions: Most NXT corpora have a layer that contains the transcriptions of speech of the participants. The {@link net.sourceforge.nite.gui.transcriptionviewer.NTranscriptionView NTranscriptionView} class can be parametrized to display these transcriptions in several ways, possibly based on existing segmentations of the transcription.
  • Display annotations defined on the transcriptions: NXT corpora usually contain several annotation layers that are directly related to the speech transcriptions, e.g. dialogue acts, named entities or referents. The NTranscriptionView class can display these annotations within the text, using configurable {@link net.sourceforge.nite.gui.transcriptionviewer.DisplayStrategy DisplayStrategy} implementations.
  • Support for selection of transcription fragments and/or annotation elements: A number of selection strategies have been implemented (speech, annotations, or both; multiple agents or single agents; single words or phrases; with or without going across segment boundaries, etc).
The best way to start using this package is to read the documentation of {@link net.sourceforge.nite.gui.transcriptionviewer.NTranscriptionView NTranscriptionView}. A table with a complete overview of the classes in this packages can be found below.
