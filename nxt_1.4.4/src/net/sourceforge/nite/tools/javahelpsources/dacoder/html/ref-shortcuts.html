Keyboard and Mouse Shortcuts

Table 1 shows the key and mouse commands used in the DA Annotation tool.

Key or mouse command Action
D Create dialogue act element
Delete Delete DA element (ask for confirmation)
Shift + Delete Delete DA element (without asking for confirmation)
R Change the range of the DA
A Create AP element
S Change the "source" of AP element
T Change the "target" of AP element
ESC Cancel creation of AP element
CTRL+S Save annotations
CTRL+P Print transcription area
CTRL + 1x Mouse-Right Play video or audio for the selected annotation
F1 Help
F5 Refresh the text area

Table 1: Key and Mouse shortcuts
